### Bank-Suite — Microservices Based Banking Application

A Java Full Stack Microservices Banking System built using Spring Boot, API Gateway, JWT Security, and React.
This project demonstrates scalable architecture, secure authentication, and real-time banking operations.

## 🚀 Features

🔐 JWT Based Authentication & Authorization
🧩 Microservices Architecture (Modular & Scalable)
🌐 API Gateway Routing
💳 Account Management (Create, Deposit, Withdraw, Transfer)
📊 Transaction History
⚡ Real-time REST APIs
🐳 Docker Support (Optional)

▶️ How to Run
1. Clone Repo
git clone https://github.com/vaibhav0809git/Bank-Suite.git

cd Bank-Suite

3. Run Backend Services

Run each microservice:
mvn spring-boot:run

3. Run Frontend
cd frontend
npm install
npm start

App runs on:
http://localhost:3000
